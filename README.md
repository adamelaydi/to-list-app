# 📝 Todo App (React)

A simple and responsive Todo application built with React.  
It helps users add, edit, delete, and manage their daily tasks efficiently.

---

## 🚀 Features

- ➕ Add new tasks
- ✏️ Edit existing tasks
- ❌ Delete tasks
- ✅ Mark tasks as completed
- 💾 Save tasks in local storage (data persists after refresh)
- 📱 Responsive UI for mobile and desktop

---

## 🛠️ Built With

- React JS (Hooks: useState, useEffect)
- JavaScript (ES6+)
- CSS / Material UI (if you used it)
- Local Storage API
